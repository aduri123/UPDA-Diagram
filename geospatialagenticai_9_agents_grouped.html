GenAI & GeoAI Agents (Reasoning & Generation)
Conversation Agent (chat/voice)
Retrieves laws, SOPs, datasets → grounds answers and tool choices. Enforces who can ask what; redacts sensitive layers; routes only allowed tool calls. LLM chain for intent → tool selection + prompt templates, memory. Orchestrates multi-step dialog → retrieve → clarify → act → summarize. In: user question; Out: grounded answer + optional action.
Planning Agent (what-if, zoning scenarios)
Pulls policies, historic plans, KPIs to justify scenario generation. Guards policy scopes, prevents disallowed simulations. Multi-tool chains (sim models, KPI calculators, report writer). Graph: generate options → score → compare → pick → explain. In: objectives/constraints; Out: ranked scenarios + rationale.
Code Agent (SQL/Cypher/Python/GP tools)
Fetches schema/docs/examples to generate correct code. Whitelists tools, schemas, DBs; masks credentials; logs execution. Tool-calling to DBs, KG, GPServer; validators in chain. Safe loop: draft → validate → dry-run → execute → rollback. In: NL request; Out: executable code + result layer.
Retrieval Agent (RAG)
Core: hybrid search over regs, specs, datasets; chunking + rerank. Ensures only permitted corpora/indexes used; audit trail for citations. Retrieval pipelines (vector/keyword), rewriters, summarizers. Branching: fail → reformulate → fallback sources → cite. In: query; Out: citations + grounded context/summary.
TestGen Agent (synthetic data & drift detection)
Retrieves sample schemas to generate synthetic test data. Validates allowed datasets for test generation. Chains for dataset synthesis, regression test automation. Scenarios: generate → inject drift → test compliance. In: schema/rules; Out: synthetic datasets, drift alerts.
GeoAI Agent (imagery, LiDAR, spatial ML)
Optionally retrieves model cards and training specs for grounding. Controls model execution permissions; tracks datasets & versions. Chains for pre/post-processing (tiling, reprojection, inference). Flow: fetch tiles → infer → vectorize → validate → publish. In: imagery/point clouds; Out: features, masks, change-detect layers.
System & Data Integration Agents (Validation & Connectivity)
Audit & Validation Agent (rules, compliance, drift)
Pulls rules (codes, IDS, schemas) + prior cases to explain findings. RBAC/ABAC gates, tamper-proof logs, MCP “allow/deny” checks. Rule-check chains, schema validators, diff/drift detectors. Checkflow: data QA → rule eval → exception handling → report. In: dataset/model/output; Out: pass/fail, diffs, compliance report.
Tool-Calling Agent (API & system integration)
RAG retrieves API specs and tool docs to enable dynamic calls. MCP authorizes which external APIs/systems can be invoked. LangChain tools wrap ArcGIS, BIM, PostGIS/Oracle, REST/GP APIs. LangGraph orchestrates sequences of multi-tool calls in workflows. In: request; Out: API response or integrated action.
SDI / Geospatial API Agent (WMS, WFS, WCS connectors)
Uses RAG to fetch metadata/catalogs for available SDI layers. MCP enforces access to sensitive spatial services and audit logs. LangChain connectors for OGC APIs and layer normalization. LangGraph coordinates retrieval of SDI services into pipelines. In: spatial query; Out: standardized service layers.
