Diagram 2 — Detailed Data Flow

1) Request Intake — User → 💬 Conversation Agent → Intent + Context
2) Orchestrate — 🔄 LangGraph selects workflow; 🛡️ MCP checks scopes
3) Retrieve — 📖 RAG fetches zoning codes, policies, socio-economic briefs
4) Plan & Simulate — 🗂️ Planning Agent → 🔌 Tool-Calling Agents invoke FastAPI/ArcGIS GPServer
5) GeoAI Checks — 🗺️ Vision/GeoAI Agent verifies overlaps, density, buffers
6) Audit & Evidence — ✅ Audit Agent applies rules; compiles logs, reports, anomaly flags
7) Decision Pack — KPIs + maps + rule hits → Dashboard & Approver
